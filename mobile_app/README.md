# El-Dahih Educational Platform - Mobile Application

تطبيق محمول احترافي لمنصة الداحي التعليمية مبني باستخدام Kivy و KivyMD

## المميزات الرئيسية

### 🎨 واجهة مستخدم احترافية
- تصميم حديث وجذاب
- دعم الثيمات المتعددة (فاتح، داكن، ذهبي)
- واجهات سلسة وسهلة الاستخدام
- أيقونات ملونة وجميلة

### 🌍 دعم اللغات
- دعم كامل للغة العربية مع RTL
- دعم اللغة الإنجليزية
- تبديل سلس بين اللغات
- نصوص عربية منسقة بشكل صحيح

### 🔒 الأمان المتقدم
- حماية الشاشة من الالتقاط
- كشف الأجهزة المخترقة (Root)
- كشف المحاكيات
- كشف VPN والوكلاء
- كشف Debugger
- تشفير البيانات الحساسة
- التحقق من توقيع التطبيق
- مراقبة أمنية مستمرة

### 🔐 التشفير والحماية
- تشفير AES-256 للبيانات
- تخزين آمن للبيانات الحساسة
- حماية الملفات المهمة
- إدارة المفاتيح الآمنة

### 📱 التوافقية
- يعمل على أندرويد 5.0+ (API 21+)
- دعم معالجات ARM و ARM64
- تحسين الأداء على الأجهزة المختلفة

### 🌐 الاتصال بالمنصة
- ربط كامل مع منصة https://mr7riko.pythonanywhere.com
- جلب المواد والدروس ديناميكياً
- مزامنة آمنة مع الخادم
- معالجة الأخطاء الذكية

## التثبيت والبناء

### 1. تثبيت المتطلبات

```bash
pip install -r requirements.txt
```

### 2. إعداد Buildozer

```bash
pip install buildozer cython
```

### 3. بناء APK

```bash
# بناء APK للتطوير
buildozer android debug

# بناء APK للإطلاق (Release)
buildozer android release
```

### 4. تثبيت على الجهاز

```bash
# تثبيت مباشر
buildozer android debug deploy run

# أو يدويًا
adb install -r bin/eldahih-1.1.0-debug.apk
```

2. اختر اللغة المفضلة

2. سيتم حذف جميع البيانات المحفوظة

## التكوين

### تغيير URL المنصة


### إضافة ترجمات جديدة

عدّل ملف `language_manager.py` لإضافة لغات جديدة

## الأمان

### نصائح الأمان

1. **لا تخزن كلمات المرور بشكل مباشر** - استخدم التخزين الآمن
2. **استخدم HTTPS دائماً** - للاتصال بالخادم
3. **حدّث المكتبات بانتظام** - للحصول على تصحيحات الأمان
4. **راجع السجلات** - للكشف عن محاولات الاختراق

### الميزات الأمنية المفعلة

- ✅ حماية الشاشة (FLAG_SECURE)
- ✅ كشف الروت
- ✅ كشف المحاكيات
- ✅ كشف VPN
- ✅ كشف Debugger
- ✅ تشفير البيانات
- ✅ التحقق من التوقيع
- ✅ مراقبة مستمرة

## استكشاف الأخطاء

### المشكلة: التطبيق لا يفتح

**الحل:**
- تأكد من تثبيت جميع المكتبات
- تحقق من السجلات: `adb logcat | grep python`
- أعد بناء APK

### المشكلة: خطأ في الاتصال بالمنصة

**الحل:**
- تحقق من الاتصال بالإنترنت
- تأكد من صحة URL المنصة
- تحقق من بيانات الدخول

### المشكلة: النصوص العربية لا تظهر بشكل صحيح

**الحل:**
- تأكد من تثبيت `arabic-reshaper` و `python-bidi`
- تحقق من الخط المستخدم
- أعد بناء التطبيق

### المشكلة: الثيم لا يتغير

**الحل:**
- أعد تشغيل التطبيق
- امسح ذاكرة التطبيق
- تحقق من ملف `theme_manager.py`

## الدعم والمساعدة

للحصول على الدعم:
- راجع السجلات في `~/.eldahih/logs/`
- تحقق من الأخطاء الشائعة أعلاه
- اتصل بفريق الدعم
